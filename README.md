# ReadLab

ReadLab 是一个基于AI的阅读实验平台，旨在帮助研究人员和学生通过实验来理解和分析阅读行为。该项目使用 Google Gemini 3 Pro Preview 进行设计和开发，结合了先进的自然语言处理技术和现代化的前端框架。

## 🎯 项目概述

ReadLab 是一个跨平台的桌面应用，支持Web、Windows和macOS，采用前后端分离架构，提供了丰富的阅读辅助功能和实验管理界面。

### 主要功能

- **AI 辅助阅读**：提供文本总结、解释和翻译功能
- **阅读材料管理**：支持多种格式的阅读材料（文本、HTML、视频、音频、图片）
- **参与者管理**：管理研究参与者和分配阅读任务
- **实时统计**：监控用户活动和阅读进度
- **个性化阅读设置**：支持字体大小、字体类型和主题切换
- **AI 生成媒体**：基于文本内容生成图像和视频
- **文本转语音**：支持有声阅读功能

## 🛠️ 技术栈

### 前端技术

- **Electron**：跨平台桌面应用框架
- **React 19**：现代化前端 UI 框架
- **TypeScript**：类型安全的 JavaScript 超集
- **Tailwind CSS 4**：实用优先的 CSS 框架
- **Vite**：快速的前端构建工具
- **Heroicons**：图标库
- **Recharts**：数据可视化库

### 后端技术

- **Python 3.10+**：后端开发语言
- **Flask**：轻量级 Python Web 框架
- **SQLAlchemy**：Python ORM 库
- **MySQL**：关系型数据库
- **PyMySQL**：MySQL 驱动
- **JWT**：JSON Web Token 认证
- **bcrypt**：密码加密库

### AI 技术

- 本项目使用了**Google Gemini 3 Pro Preview**

## 📖 使用指南

### 管理员功能

1. **登录管理台**：使用管理员账号登录
2. **管理材料**：创建、编辑和分配阅读材料
3. **管理参与者**：查看和管理研究参与者
4. **查看统计数据**：监控用户活动和阅读进度
5. **管理表单**：创建知情同意书和问卷

### 参与者功能

1. **登录系统**：使用分配的账号登录
2. **阅读材料**：访问分配的阅读材料
3. **使用 AI 助手**：提问、总结或翻译文本
4. **个性化设置**：调整阅读偏好
5. **生成媒体**：基于文本生成图像和视频
6. **完成阅读**：标记材料为已阅读

## 🔧 开发流程

### 前端开发

- 代码风格：遵循 React 和 TypeScript 最佳实践
- 组件设计：使用函数组件和 Hooks
- 状态管理：使用 React Context API 或自定义 Hooks
- 路由管理：使用 React Router

### 后端开发

- API 设计：RESTful API 规范
- 数据库操作：使用 SQLAlchemy ORM
- 认证授权：JWT 认证
- 错误处理：统一的错误响应格式

### 数据库开发

- 数据迁移：使用 SQLAlchemy 迁移工具
- 数据备份：定期备份数据库
- 性能优化：合理设计索引和查询
